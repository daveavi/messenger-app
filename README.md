### Env Variables

Create a .env file in then server folder and add the following

```
NODE_ENV = development
PORT = 5000
MONGO_URI = your mongodb uri
```

### Install Dependencies (frontend & backend)

```
cd server
npm install

cd client
npm install
```

### Run

```
# Run frontend (:3000) 
npm start

# Run backend backend (:5000)
npm run dev
```
